# Yusuf Öz — Portfolio

Interactive portfolio terminal for Yusuf Öz. Software Engineer focusing on Mobile, Web, and Autonomous Systems.

## Tech Stack
- **Framework:** Next.js 14 / React 18
- **Styling:** Tailwind CSS + Custom CSS Variables
- **Animations:** Framer Motion, GSAP, Studio Freight Lenis
- **Typography:** Instrument Sans, Space Grotesk, VT323

## Structure
- App Router (`app/`)
- Reusable UI chunks (`components/ui`)
- Parallax / Smooth Scroll engine (`components/SmoothScroll.tsx`, `components/ParallaxNebulae.tsx`)

## Deployment
Responsive and optimized for deep space.
